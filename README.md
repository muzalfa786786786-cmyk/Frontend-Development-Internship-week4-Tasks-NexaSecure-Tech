# 👠 Jutti Heels – Fashion E-Commerce Website

A fully responsive, multi-page fashion e-commerce website for a modern heels and footwear brand.  
This project was developed using **HTML5, CSS3, and JavaScript** during my **Frontend Development Internship at NexaSecure Tech**.

The website features a clean and elegant user interface where users can explore products, view product details, select colours and quantities, and interact with modern UI components. The design is optimized for **desktop, tablet, and mobile devices**.

# ✨ Features

• Fully responsive design (Mobile, Tablet, Desktop)  
• Multi-page website (Home, Products, About, Contact, Product Details)  
• Interactive product cards with hover effects  
• Add to Cart functionality with notifications  
• Product details page with image gallery  
• Colour selection with visual feedback  
• Quantity selector for product purchase  
• Mobile-friendly navigation menu  
• Contact form with input validation  
• Smooth UI animations and transitions  

---

# 🛠 Technologies Used

• **HTML5** – Semantic webpage structure  
• **CSS3** – Flexbox, Grid layout, responsive design, animations  
• **JavaScript (ES6)** – DOM manipulation and form validation  
• **Google Fonts** – Playfair Display & Inter  
• **Font Awesome** – Icons  
• **Netlify** – Website deployment and hosting
