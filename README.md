# 👠 Jutti Heels – Fashion E-Commerce Website

A fully responsive, multi-page fashion e-commerce website for a modern heels and footwear brand.  
This project was developed using **HTML5, CSS3, and JavaScript** during my **Frontend Development Internship at NexaSecure Tech**.

The website features a clean and elegant user interface where users can explore products, view product details, select colours and quantities, and interact with modern UI components. The design is optimized for **desktop, tablet, and mobile devices**.
